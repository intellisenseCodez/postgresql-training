
# Day 3 Command Reference

## Core SQL Clauses

- SELECT
- DISTINCT
- DISTINCT ON
- FROM
- WHERE
- ORDER BY
- LIMIT
- OFFSET
- FETCH
- GROUP BY
- HAVING

## Comparison & Logical Operators

- =, !=, <, >, <=, >=
- BETWEEN, IN, LIKE
- AND, OR
- IS NULL, IS NOT NULL

## Aggregate Functions

- COUNT() – number of rows
- SUM() – total value
- AVG() – average value
- MIN() – smallest value
- MAX() – largest value

## PostgreSQL-Specific Feature
- DISTINCT ON (column) – first row per group (based on ordering)
