# PostgreSQL Bootcamp: A Free 12-Week SQL Focused Trainning

<p style="text-align: center;"> Master PostgreSQL fast through many practical examples and apply the knowledge to application development using PostgreSQL.</p>

[LaunchPad with CoachOla](https://chat.whatsapp.com/IseaP1Hus1j0I9Qy7mVexu)

## Bootcamp Overview

**Goal:** Take learners from zero to advanced PostgreSQL skills, including real-world use cases, performance optimization, and production-ready practices.

**Target Audience:** Engineers, Analysts, Data Professionals, Backend Developers.

**Structure:** Each week contains:
- Learning objectives
- Slides / lesson notes
- Hands-on labs
- Homework assignments
- Reference solutions

## Prerequisites

- Basic SQL knowledge (optional but helpful)
- Installed PostgreSQL (v14+ recommended)
- Familiarity with command line basics
